Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The starting bracket begins with a <"code"> symbol.
  • The end bracket ends with a <"/code"> symbol.
  • The html lang="en" shows the webpage is in english.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS helps to define how web content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organisation.
  • You can style a webpage inline CSS, internal CSS style sheet, and external CSS style sheet.
  • An example of inline CSS is using the "<"style"> element within the code.
  • The most common way to style a webpage is by using an external CSS style sheet and linking the file in the code using a .

Git

  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creates a new branch and switches to it.
  • Use "git" to access the git commands.
  • Git add, commit and push are essential steps for developers using github!
  • A git repository is connected to the local working directory through the SSH code.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • Developers have written a variety of tools on top of the core JavaScript language, unlocking a vast amount of functionality with minimum effort.
  • The reason the instructions (above) place the <"script"> element near the bottom of the HTML file is that the browser reads code in the order it appears in the file.
  • Variables are containers that store values. You start by declaring a variable with the let keyword, followed by the name you give to the variable: "let myVariable;"
  • You can name a variable nearly anything. They are case sensitive.
